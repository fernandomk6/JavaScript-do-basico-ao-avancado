O que são funções

  • Funções são estruturas de códigos menores que são reaproveitadas durante a execução/construção de um programa
  • Principal objetivo: Poupar repetição de código
  • Podem ser consideradas 'sub-programadas'

  • Exemplo de declarações de funções

    1.                         function escreverNoConsole(){
                            console.log("Escrevendo no console!");
                        }

                        escreverNoConsole();
                    
    2.                         const textNoConsole = function(){
                            console.log("Texto no console!");
                        }

                        textoNoConsole();
                    
    3.                         const textPorParametro = function(text){
                            console.log(text);
                        }

                        textPorParametro("Texto por parâmetro");
