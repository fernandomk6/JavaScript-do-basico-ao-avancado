Strings: detalhes mais técnicos

  • A \ (contra barra) pode dar um 'escape', e isso permite algumas coisas especiais
  • Por exemplo \n pula uma linha
  • console.log("Essa é uma String \n De duas linhas")
  • Para inserir uma ' ou " devemos iniciar a string com as aspas inversa
  • O template literals serve para computar valores também
  • console.log(´A soma de 2 + 2 é ${2 + 2}´)
  • concatenação é um processo de somar strings
  • console.log("salada"+" de "+"frutas")
