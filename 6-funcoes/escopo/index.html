Escopo

  • O que acontece dentro de uma função, fica serapara do escopo global
  • Escopo global, seria todo o arquivo javascript

  • Exemplo

            let n = 10;
        function teste(){
          let n = 25;
          console.log(n);
        }
        teste();
        console.log(n);

        // perceba que as variaveis n, exibirão valores diverentes, respeitando o escopo o qual foram declaradas
  • Usando let e const também são respeitado os escopos de if e else e blocos de codigos em geral e não so apenas em funções
