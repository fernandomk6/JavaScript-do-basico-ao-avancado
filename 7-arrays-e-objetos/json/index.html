JSON

  • JSON = JavaScript Object Notation
  • Utilizado para comunicação entre serviços, ex: back end <-> front end
  • Não possui metodos apenas propriedades, as chaves são entre aspas duplas, aceita arays e outros json dentro do json
  • Basicamente um tipo de dado padronizado, que lembra muito os objetos do javascript
  • Os arrys devem ser de um tipo de dado apenas
  • Tipos de dados string, numero, booelan, null
  • não possui metodos
